# Dogger App

Dogger App is a modern Vue 3 application that displays cute dog breeds and random dog images using the Dog CEO API.

## Features

- Vue 3 with Composition API
- Vue Router navigation
- Breed overview page
- Breed detail page
- Random dog images
- Smooth page transitions
- Smooth image loading animations
- Loading and error states
- Responsive card layout
- TanStack Query for data fetching and caching
- Cached breed list
- Cached breed images during the same session
- Naive UI component library

## Technologies

- Vue 3
- Vue Router
- TanStack Query
- Naive UI
- Vite

## API

This project uses the Dog CEO API:

- https://dog.ceo/api/breeds/list/all
- https://dog.ceo/api/breed/{breed}/images/random

## Setup

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

## Production Build

```bash
npm run build
```
